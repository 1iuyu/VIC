CESM Driver for VIC Model
================

This driver was developmed for coupling VIC within the Regional Arctic System
Model (RASM).

# Directory Structure
    
    | - cesm
    | --- bld
    | --- cpl_esmf
    | --- cpl_mct
    | --- include
    | --- src

# Building the Model
The `bld/` directory contains the build scripts used by the CESM `$case.build`
script.
